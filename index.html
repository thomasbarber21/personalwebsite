Projects

  • Income and Education: A Cross Regional Look into the Impact of Income and Education

    This econometric research used a dataset containing millions of inputs from the IPUMS database. Using R, I built linear regression models and analyzed trends between variables including gender, education, age, and region of residence. All this was used to explore the relationship between such factors and income.

  • Dimensionality Reduction using Probabilistic Principal Component Analysis

    This research project reviewed a probabilistic approach to principal component analysis and its contribution to dimensionality reduction. The project then used this concept to recreate data from the famous MNIST dataset as well as exploring simple Gaussian Mixture Models.

  • A Review of Linear Algebra Applications in Image Processing

    This project is a review paper on different techniques of image capture, the application of linear algebra within such techniques, and its application to food quality analysis.

  • Advanced Data Visualization Techniques in R

    This project explored the world of data visualization in R. Specifically, I did a deep dive into both ggplot and plotly packages. I then compared both methods to the standard plotting methods in R to find the pros and cons of each method and determined when each one is best used.
